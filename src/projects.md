## Research Tools

<div class="project-card">

### [P(Doom) Suite](https://github.com/PipFoweraker/pdoom1)
<span class="project-status active">Active</span>

A set of tools for tracking and visualising existential risk probability estimates. Includes the core methodology ([pdoom1](https://github.com/PipFoweraker/pdoom1)), structured datasets ([pdoom-data](https://github.com/PipFoweraker/pdoom-data)), an interactive dashboard ([pdoom-dashboard](https://github.com/PipFoweraker/pdoom-dashboard)), and a public website ([pdoom1-website](https://github.com/PipFoweraker/pdoom1-website)).

</div>

<div class="project-card">

### [Interrogatory Model Cards](https://github.com/PipFoweraker/interrogatory-model-cards)
<span class="project-status experimental">Experimental</span>

A proposal for interrogatory, low-friction model cards using the CAN/SHOULD/MUST framework. Designed to make AI model documentation more actionable and less performative.

</div>

<div class="project-card">

### [DoubleGiniDoubleLorenz](https://github.com/PipFoweraker/DoubleGiniDoubleLorenz)
<span class="project-status experimental">Experimental</span>

Statistical tool for creating and mapping dual Gini coefficients and Lorenz curves from different data sources to infer bounds on position estimates.

</div>
